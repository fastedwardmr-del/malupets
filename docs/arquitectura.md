# Arquitectura Malupets

## MVP

1. Core: empresa, usuarios, configuración.
2. Clientes y mascotas.
3. Inventario y servicios.
4. POS, ventas, pagos y caja.
5. Agenda.
6. Reportes básicos.

## Convenciones

- Worker: `api-malupets`
- D1: `malupets-db`
- Binding D1: `DB`
- R2: `malupets-files`
- Binding R2: `FILES`
